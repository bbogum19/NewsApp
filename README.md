# News-App Nedir?
Gündemdeki olayları mobilde görüntüleyebileceğiniz bir **haber** uygulamasıdır. Ekranda **haberin resmi** , **haberin açıklaması** ve **haberin yazarı** bilgileri yer almaktadır.
